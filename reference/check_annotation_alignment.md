# Check AnnDatR Alignment Verifies that the expression matrix (X) rows and columns are correctly ordered and aligned with the gene (var) and cell (obs) metadata tables.

Check AnnDatR Alignment Verifies that the expression matrix (X) rows and
columns are correctly ordered and aligned with the gene (var) and cell
(obs) metadata tables.

## Usage

``` r
check_annotation_alignment(adata)
```

## Arguments

- adata:

  AnnDatR object.

## Value

Logical TRUE if alignment is perfect, FALSE otherwise.
