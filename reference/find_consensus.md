# Finds consensus clustering from multiple clusterings

Finds consensus clustering from multiple clusterings

## Usage

``` r
find_consensus(all_clusterings, n, seed = 42, verbose = TRUE)
```

## Arguments

- all_clusterings:

  Data frame containing clustering results from multiple runs

- n:

  Number of clustering runs

- seed:

  Random seed for consensus clustering

- verbose:

  Logical indicating whether to print progress messages

## Value

Consensus clustering results
